Os Sons da Letra X
A letra X tem 4 sons diferentes em português! Toque nas palavras para ouvir.
X = SHX = S X = KSX = Z
⚠️ Atenção
O X brasileiro é muito diferente do espanhol! Em espanhol X quase sempre soa como KS. Em português tem 4 sons diferentes.
Como aprender
1. Aprenda cada som · 2. Toque para ouvir · 3. Repita em voz alta · 4. Faça as atividades
🔤 Os 4 Sons do X
X = /SH/
Mais comum!
Início da palavra ou após ditongo (ai, ei, ou...)
xícara ☕ xale 🧣 roxo 🟣 caixa 📦 peixe 🐟 embaixo ⬇️
X = /S/
Muito frequente
Prefixo "ex-" antes de consoante, e algumas outras
máximo próximo trouxe texto extensão
X = /KS/
Como espanhol
Palavras de origem grega ou latina
táxi 🚕 fixo tórax reflexo
X = /Z/
Cuidado!
Prefixo "ex-" antes de vogal: exa-, exe-, exi-...
exame 📋 exemplo exato exagero
⚡ Casos Especiais
Depois de ditongo → sempre /SH/
caixa peixe frouxo faixa
CH = /SH/ — mesmo som!
Em português, ch soa igual a X=/SH/
chave 🔑 chocolate 🍫 chuva 🌧️
💡 Dica rápida
X no início → quase sempre /SH/ · "ex-" + vogal → /Z/ · Para o resto, aprenda palavra por palavra!
👅 Trava-língua
O Xico xingou o chico,
o chico chorou.
A xícara de chá chegou,
o Xico se animou.
Como praticar
1. Ouça · 2. Repita devagar · 3. Acelere · Todas as letras destacadas são /SH/
